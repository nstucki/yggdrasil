# Performance Analysis

## Purpose

Analyze systems for performance issues and optimization opportunities.

## Instructions

- Identify potential bottlenecks.
- Analyze resource usage and execution paths.
- Evaluate scalability concerns.
- Suggest measurable improvements.
- Consider tradeoffs between performance and complexity.

Focus on diagnosis and recommendations, not implementation.
