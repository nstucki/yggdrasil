# Documentation Research

## Purpose

Gather and analyze relevant documentation and external knowledge.

## Instructions

- Locate authoritative documentation sources.
- Extract relevant concepts and constraints.
- Compare documentation with current implementation.
- Summarize important findings.
- Highlight version-specific considerations.

Focus on providing reliable information for decision-making.
