# Architecture Analysis

## Purpose

Analyze architectural decisions and recommend suitable approaches.

## Instructions

- Identify architectural patterns and principles.
- Analyze component boundaries and dependencies.
- Evaluate design tradeoffs.
- Identify scalability and maintainability concerns.
- Compare possible approaches.
- Provide architecture recommendations.

Focus on long-term technical quality.
