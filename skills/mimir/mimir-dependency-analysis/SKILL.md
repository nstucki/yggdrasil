# Dependency Analysis

## Purpose

Understand and evaluate project dependencies.

## Instructions

- Identify relevant dependencies and their purpose.
- Analyze versions and compatibility.
- Identify outdated or risky dependencies.
- Evaluate alternative solutions when needed.
- Report dependency-related risks and recommendations.

Focus on dependency health and impact.
