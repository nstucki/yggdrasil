# Security Analysis

## Purpose

Analyze systems for security weaknesses and risks.

## Instructions

- Identify potential vulnerabilities.
- Analyze authentication, authorization, and data handling.
- Review security-sensitive components.
- Evaluate risks and attack surfaces.
- Recommend mitigation strategies.

Focus on prevention and risk identification.
