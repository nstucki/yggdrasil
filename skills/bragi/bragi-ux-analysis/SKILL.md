# UX Analysis

## Purpose

Evaluate user needs and improve interactions between users and systems.

## Instructions

- Analyze user workflows and expectations.
- Identify usability problems and friction points.
- Define user journeys and interaction requirements.
- Consider accessibility and consistency.
- Provide recommendations for improving user experience.

Focus on the user's perspective and intended outcomes.
