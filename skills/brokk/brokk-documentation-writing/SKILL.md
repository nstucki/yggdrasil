# Documentation Writing

## Purpose

Create clear and useful documentation.

## Instructions

- Write documentation based on project needs.
- Explain usage, architecture, and processes.
- Keep documentation accurate and maintainable.
- Follow existing documentation conventions.
- Adapt explanations to the intended audience.

Focus on clarity and long-term usefulness.
